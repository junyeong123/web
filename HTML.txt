HTML
-제일 위에 관용적으로 <!doctype html>을 쓴다.
-<!doctype html> 다음줄에 <html>을 쓴다. 본문 마지막에는 </html>를 쓴다.
<html>다음에는 <head>쓴다. </head>은 타이을 마지막에 쓴다.
-제목 부분에 <tittle>을 쓴다. </tittle>도 써야한다.
-<meta charset="utf-8">은 문서의 형식을 <meta charset="utf-8">이렇게 저장한다라는 약속이다.
-본론 부분으로 넘어가서는 <body>을 위에 쓴다. 본론 마지막에는 </body>쓴다.
-<ol>,<ul>은 목차의 순서를 매길때 쓴다.
-<li>는 리스트를 만들때 쓴다.
-<h1~9>는 글씨의 크기를 조절할때 쓴다.
-<p>는 문단을 나눌때 쓴다.
-<br>은 줄을 바꿀때 쓴다. -> </br>은 쓸필요 없다.
<img src = "qwe.jpg" width = "200">에서 img는 이미지를 넣는다는 의미이고 scr은 파일명(사진명) width은 크기를 나타낸다.
-<a>는 링크를 걸때 쓴다 
-<a>에서 href = "주소", target="blank"는 새탭으로 열려고 할때 쓴다. 링크를 클릭하기전 툴팁으로 미리 알려주려면 tittle="~"
